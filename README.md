# Task Tracker (React+Tailwind CSS)

Simple Task Tracker web application usinf **React.js**, **TailwindCSS**,**UUID**, **React Toast** and **Local Storage**.

## [Live Demo](https://task-tracker-irip.onrender.com)

## Features

* **Add, Edit and Delete Tasks**
* **Track Complete and Pending Tasks**
* **Filter Tasks (All, Pending, Completed)**
* **Local Storeage to Store Tasks**
* **UUID for unique id generation of each tasks to operate on**
* **Interactive UI with React Toastify based Notification**

## Setup and Installation

### Clone the Repository

```bash
git clone https://github.com/Vikram-353/Task-Tracker-
```

### Install Dependencies

```bash
npm install
```

### Setup env for localStorage

1. Create .env file
2. Copy variable form .env.demo file and paste it in .env file

### Run Application

```bash
npm run dev
```
